# 🤖 AI Tools Hub

A fully responsive, modern multi-page website built with HTML, CSS, and JavaScript for showcasing:

- 📊 Confusion Matrix Generator
- 🧾 Classification Report Generator

## 🔗 Live Tools

- 📊 Confusion Matrix Generator: `https://confusion-matrix-generator.vercel.app/`
- 📈 Metrics Table Generator: `https://ml-metrics-table-generator.vercel.app/`

## 📄 Pages

- `index.html` (Home)
- `about.html` (About)
- `contact.html` (Contact)

## ✨ Features

- 📌 Sticky responsive navbar with mobile hamburger menu
- 🦸 Hero section and animated tool cards
- 🪟 Tool modals on Home page
- 🎓 Beginner-friendly About content with icons
- 📬 Contact form with JavaScript validation and success message
- 🎬 Fade-in scroll animations
- ⏳ Loading animation
- ⬆️ Scroll-to-top button
- 🌙 Dark mode toggle in navbar
- 📱 Mobile-first responsive layout (mobile/tablet/desktop)

## 🗂️ Project Structure

```text
index.html
about.html
contact.html
css/style.css
js/script.js
```

## 🚀 Run Locally

You can simply open `index.html` in your browser, or run a lightweight local server:

```powershell
Set-Location "c:\Users\ai-tools-hub"
python -m http.server 5500
```

Then open:

- 🌐 `http://localhost:5500/index.html`
