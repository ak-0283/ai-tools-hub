What is a Confusion Matrix?

A confusion matrix is a table used to check how well a classification model is performing. It compares what the model predicted with the actual correct answers.

In simple words, it helps you see where your model is right and where it makes mistakes.

🧾

What is a Classification Report?

A classification report gives detailed performance numbers like precision, recall, f1-score, and support for each class.

It helps you understand not just if your model works, but how well it performs for each category.
